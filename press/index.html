Short Description
Promptlyss is a precision-thinking game where players study an AI-generated image and choose the exact prompt that produced it. Unlock packs, build streaks, and climb leaderboards.
Key Features
  • Prompt-guessing gameplay based on visual reasoning
  • Packs & progression (coins / StarXP)
  • Daily challenge and streak system
  • Leaderboards
  • Multiple difficulty tiers
